# Professional Email Drafter

Draft a professional email based on the context below.

## Context

- **From:** {{SENDER_ROLE}}
- **To:** {{RECIPIENT_ROLE}}
- **Relationship:** {{RELATIONSHIP}}
  > e.g. "manager to direct report", "vendor to prospective client"
- **Goal:** {{EMAIL_GOAL}}
  > e.g. "Request a project deadline extension", "Follow up on unpaid invoice"
- **Tone:** {{TONE}}
  > e.g. "Formal and direct", "Warm but professional"
- **Key facts to include:** {{KEY_FACTS}}

## Instructions

1. Write a concise subject line (≤60 characters).
2. Open with a context-setting sentence — no filler like "Hope this finds you well".
3. State the purpose clearly in the first paragraph.
4. Use short paragraphs (2–3 sentences max).
5. End with a clear call to action and deadline if relevant.
6. Sign off appropriately for the `{{TONE}}`.
