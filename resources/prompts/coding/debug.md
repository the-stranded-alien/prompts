## Debugging Process

### 1. Reproduce
Confirm you understand the exact steps to reproduce the bug and the expected vs actual behaviour.

### 2. Hypothesise
Generate a ranked list of possible root causes based on the symptoms.

### 3. Investigate
For each hypothesis, describe:
- What evidence supports or refutes it
- What additional information would confirm it

### 4. Isolate
Narrow down to the single most likely root cause with evidence.

### 5. Fix
Provide the minimal code change that resolves the root cause. Avoid patching symptoms.

### 6. Prevent
Suggest a test or guard that would catch this class of bug in the future.
